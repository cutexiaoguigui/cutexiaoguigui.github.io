# 龟之飞  

## 介绍

使用hexo搭建的blog网站，可以对文章评论、搜索文章、查看动态等。采用了响应式布局，使得网站可以自适应不同的屏幕大小，从而在桌面电脑、平板电脑和智能手机上都能够获得最佳的视觉效果和用户体验。🐣

⭐我们的网站地址是：**https://cutexiaoguigui.eu.org/** ，备用站地址为：**https://backup.cutexiaoguigui.eu.org/** 。  

> 1. 我们有许多方面的研究，有blog，ai，网盘等。
> 2. 见后续blog文章中有具体说明。
